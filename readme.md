# 安装依赖
pip install pillow

# 使用说明
将 font-awesome-to-png.py 放到 font-awesome 主目录下
运行 font-awesome-to-png.py
可接受参数 size=int color=#str
不输入参数默认导出 50x50 黑色字体图片
示例：font-awesome-to-png.py size=100 color=#ff00ff
